Bet365是一家成立于 2000 年的英国赌博公司。其产品包括体育博彩、在线赌场、在线扑克和在线宾果游戏。业务运营在特伦特河畔斯托克的总部进行，并在马耳他设有卫星办事处。该公司由丹尼斯·科茨 (Denise Coates)创立，她与她的兄弟约翰·科茨 (John Coates)仍然是大股东兼联席首席执行官。

Bet365 是一家在线赌博公司，提供体育博彩和赌场类型的游戏。

除了位于特伦特河畔斯托克的公司总部外，Bet365 在曼彻斯特、直布罗陀、[1] 马耳他、[2] 保加利亚和澳大利亚还设有办事处。截至 2020 年，该集团雇用了 4,000 多名员工。[3] [4]

Bet365 是 Hillside (New Media) Ltd. [5]和运营商的商业名称。

Bet365 于 2000 年由Denise Coates在特伦特河畔斯托克的一座可移动建筑[3]中创立。丹尼斯开发了一个体育博彩平台和交易团队，并于 2001 年 3 月在网上开展业务。该公司以家族博彩店庄园为抵押，从苏格兰皇家银行借款 1500 万英镑，该庄园由彼得·科茨 (Peter Coates) 于 1974 年创办，并由丹尼斯经营。科茨从 1995 年起担任董事总经理。Bet365 于 2005 年以 4000 万英镑的价格将其博彩连锁店出售给Coral，并偿还了苏格兰皇家银行的贷款。[6]

Bet365 董事长彼得·科茨(Peter Coates)也是斯托克城的主席，2012 年 5 月，Bet365 与俱乐部签署了一份为期三年的合同，成为球衣赞助商。[7] 2016年4月，该公司取代当地企业不列颠尼亚建筑协会，成为俱乐部球场未来六个赛季的新冠名赞助商。[8] 2016年夏天，Bet365还与保加利亚俱乐部Ludogorets Razgrad和Slavia Sofia签署了未来两个赛季的球衣赞助协议。[9] [验证失败]

Bet365 的报告数据（2020 年 3 月至 21 日）显示收入为 28 亿英镑，税前利润为 4.7 亿英镑。[10]

联合首席执行官丹尼斯·科茨 (Denise Coates) 继续运营 Bet365，并是持有 50.1% 股份的大股东。她的兄弟约翰 (John)担任联合首席执行官，与她一起经营公司，而她的父亲彼得 (Peter) 担任董事长。

2019年夏天，英国最大的博彩公司和在线赌场运营商William Hill、GVC Holdings、Flutter Entertainment、Stars Group和Bet365签订了一项协议，转移资金以打击赌博成瘾。他们同意在未来五年内将这一数额从总收入的0.1%增加到1%。该倡议受到了减少危害组织的欢迎，但也因其不足而受到批评；一位议员将其描述为“为了安抚活动人士和英国政府而进行的贿赂”。[11]
